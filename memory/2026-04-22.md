### 🐛 问题与解决方案记录
1. **问题**：代码提交后用户看到还是旧页面
   - **原因**：Cloudflare CDN节点缓存延迟，部分地区节点还返回旧版资源
   - **解决方案**：用户用隐私模式访问/按Ctrl+Shift+R强制刷新；部署后主动清除CDN全量缓存

2. **问题**：部分页面导航栏还是旧样式
   - **原因**：早期开发的页面（dashboard、warning）硬编码了导航栏，未替换为全局组件
   - **解决方案**：全量排查所有页面，统一替换为`Navbar`组件，后续新页面全部使用全局组件

3. **问题**：构建失败
   - **原因**：组件导入路径错误，或者未声明变量
   - **解决方案**：提交前本地运行`npm run build`验证构建成功再提交

### 📌 注意事项
1. 所有新页面必须使用全局`Navbar`组件，禁止硬编码导航
2. 提交代码前必须本地运行`npm run build`验证构建成功，防止线上部署失败
3. 功能变更后必须自己用隐私模式访问验证通过再通知用户验收
