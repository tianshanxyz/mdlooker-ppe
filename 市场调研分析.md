# PPE合规与撮合服务市场调研分析

## 1. 市场规模与增长驱动
- **PPE认证市场**：2024年规模73.7亿美元，预计2035年达125亿美元，CAGR约4.9%。
- **核心驱动**：全球监管收紧（GPSR深化、EU PPE Regulation持续更新），中国制造企业出海面临更严苛的准入门槛。
- **合规人才缺口**：RAPS 2025年会员仅18,500人，工作量增长10.52%，合规人才严重短缺。

## 2. 竞品格局深度剖析

### 2.1 Euverify
| 维度 | 信息 |
|------|------|
| 定位 | 一站式产品合规SaaS，覆盖98个品类、5,200+产品 |
| 核心功能 | 24h欧代/英代指定、技术文件生成器、DoC生成器、平台对接(Amazon/eBay) |
| 定价 | £39/月起 |
| 客户规模 | 5,000+品牌，Trustpilot 4.7/5 |
| **核心弱点** | PPE非核心品类，深度有限；纯SaaS无人工服务；无供应链能力 |

### 2.2 Pure Global
| 维度 | 信息 |
|------|------|
| 定位 | AI驱动的医疗器械法规咨询平台，覆盖30+国家 |
| 核心产品 | AI Builder文档生成、GRIP法规智能平台 |
| 定价 | 固定年费制，起价$2,000/年/设备 |
| **核心弱点** | 主攻医疗器械(MDR/IVDR)，PPE非战略重点；价格门槛高，排除中小客户 |

### 2.3 RegDesk
| 维度 | 信息 |
|------|------|
| 定位 | AI驱动的RIMS平台(医疗器械、制药、IVD) |
| 核心功能 | AI翻译、变更评估、标准管理、EUDAMED集成 |
| 用户评价 | Ease of use 5.0, Value for money 5.0 |
| **核心弱点** | 企业级定价，中小客户无法触及；需询价 |

### 2.4 MEDPilot
| 维度 | 信息 |
|------|------|
| 定位 | 台湾国研院开发，AI驱动的医疗器械法规平台 |
| 技术特点 | LLM+RAG+推理链，搭配FDA开放数据库 |
| **参考价值** | RAG架构的法规问答系统技术路线 |

### 2.5 国内传统认证代理（如上海湘应等）
- **模式**：全案委托式代办，人肉中介。
- **短板**：无技术能力、效率低、服务不透明、无供应链转化能力。

### 2.6 B2B撮合平台（Alibaba/Global Sources）
- **模式**：海量供应商与买家撮合。
- **短板**：品类泛化、无合规验证基因、买家筛选成本高。

## 3. 差异化机会总结
| 竞争者 | 核心弱点 | 你的差异化切入 |
|--------|---------|--------------|
| Euverify | PPE非核心，无人工，无供应链 | **PPE垂直深度 + 人工增值服务 + 供应链闭环** |
| Pure Global | 主攻医材，价格高 | **专注PPE，低价年费制，服务中小买家** |
| RegDesk | 企业级定价 | **$199-$499/年，大幅降低使用门槛** |
| 传统代办 | 无技术，效率低 | **数字化平台提升透明度与效率** |

**核心结论**：市场中不存在同时具备“PPE垂直深度 + 数字化能力 + 合规服务 + 生产供应链”的竞争者。该生态位真实存在。
