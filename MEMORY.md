# MEMORY.md - Long-Term Memory

## Preferences

- **联网搜索优先使用 searxng skill** —— 只要涉及联网搜索任务，优先调用 searxng 技能而非直接使用 web_search 工具。

## 记忆管理规则
1. 每日必须创建对应日期的记忆文档（memory/YYYY-MM-DD.md），有内容则详细记录，无内容则简短标注
2. 每天第一次对话开启时，自动查看最近三天的记忆文档和长期记忆文档（MEMORY.md）
3. vault文件夹用于存储体积较大、不便直接存入记忆文档但需要留存的文件，只需在记忆文档中留存对应索引即可
