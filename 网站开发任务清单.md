# mdlooker PPE合规平台开发任务清单

## 1. 项目概述
- **项目名称**：mdlooker PPE Compliance Toolkit
- **技术栈**：Next.js + Tailwind CSS (前端)，Supabase (后端/数据库)，OpenAI API + Pinecone (AI问答)，Vercel (部署)
- **目标**：将现有mdlooker改造为面向全球PPE产业链用户的垂直合规SaaS工具包。

## 3. 定价与获客功能设计
| 方案 | 年费 | 核心功能 | 获客策略 |
|------|------|---------|---------|
| 免费版 | $0 | 基础法规查询、自检工具(限3次/月)、邮件订阅 | 作为引流入口，收集销售线索 |
| Pro版 | $199 | 无限次自检、DoC生成器、模板库下载、法规追踪 | 对LinkedIn咨询客户赠送1个月试用 |
| Enterprise版 | $499 | Pro全部功能+文档协作平台+专家对接优先通道+1次免费合规咨询 | 对参展客户或高意向客户赠送3个月试用 |

## 4. 非功能性需求
- **多语言**：界面与内容以英文为主。
- **SEO优化**：关键页面(如PPE品类自检页)针对“PPE CE certification requirements”等长尾词优化。
- **响应式设计**：完美适配移动端与桌面端。
- **数据安全**：用户上传的产品信息加密存储。

## 5. 部署与维护
- **代码托管**：GitHub Private Repository。
- **部署平台**：Vercel (与Next.js无缝集成)。
- **数据库**：Supabase (PostgreSQL，提供免费额度)。
- **域名**：mdlooker.com (已有)。
- **月均运营成本预估**：$20-$50 (主要为OpenAI API调用费)。
