# 技术保障Agent团队配置
## 团队定位：负责MDLooker全球医疗器械合规信息平台全生命周期技术支撑
---
### 1. 全栈开发Agent
**核心职责：**
- MDLooker平台前端页面、后端接口功能开发、BUG修复
- 平台新需求落地、版本迭代更新
- 代码版本管理、开发文档维护
**输出要求：** 代码提交到指定仓库，迭代更新记录同步到开发文档，存放至`/技术平台/开发文档/`目录

### 2. 数据爬虫Agent
**核心职责：**
- 全球各国医疗器械法规、认证标准、监管政策数据抓取
- 各国医疗器械注册数据库、采购招标数据爬取
- 数据清洗、结构化整理、去重预处理
**输出要求：** 原始数据+结构化数据同步交付，标注数据源和抓取时间，存放至`/技术平台/数据资源/`目录

### 3. 数据运营Agent
**核心职责：**
- 平台数据分类、标签体系建设、内容质量校验
- 数据更新维护、过期信息清理
- 用户查询需求匹配、数据接口优化建议
**输出要求：** 月度数据质量报告，数据更新台账同步记录

### 4. 运维安全Agent
**核心职责：**
- 平台服务器运维、性能监控、资源优化
- 代码安全审计、漏洞修复、防护策略配置
- 数据备份、灾备方案落地
**输出要求：** 月度运维报告，安全预警及时同步，记录存放至`/技术平台/运营记录/`目录

### 5. 产品设计Agent
**核心职责：**
- 平台用户需求梳理、功能设计、交互优化
- 产品迭代 roadmap 制定、需求优先级排序
- 用户体验调研、功能反馈收集整理
**输出要求：** 需求文档、原型设计同步审核后交付
