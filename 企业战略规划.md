# PPE合规与撮合业务企业战略规划

## 1. 公司定位与品牌
- **公司形态**：一人公司，远程运营，轻资产，面向全球市场。
- **品牌名称**：MedReg Bridge（备选：ComplyBridge）
- **品牌定位**：PPE产业链的合规工程师与全球市场连接器。
- **核心 slogan**：From Factory to Global Market — One Bridge.

## 2. 业务架构与盈利模式

### 2.1 三层业务结构
| 层级 | 业务类型 | 盈利模式 | 目标 |
|------|---------|---------|------|
| 获客层 | 合规信息平台(mdlooker) | 免费工具+低价年费($199-$499/年) | 获取海量销售线索，建立专业信任 |
| 转化层 | 合规代办服务(CE/FDA/UKCA) | 服务费差价($3,000-$15,000/单) | 深度变现，建立客户关系 |
| 放大层 | 外贸撮合/生产订单对接 | 订单佣金(3-5%)或工厂利润分成 | 将客户LTV提升5-10倍 |

### 2.2 供应链闭环模式
> **合规平台引流 → 代办服务建立信任 → 自然过渡至家族工厂OEM订单**

## 3. 阶段性执行路线图

### 第一阶段：0-3个月——外贸获客筑基（现金流引擎）
- **目标**：获取1-3个外贸看厂客户，实现正现金流。
- **动作**：
  - 专业官网搭建（已完成）。
  - LinkedIn精准获客（每日连接10-20人，提供合规价值内容）。
  - 内容营销：每周1篇PPE合规/采购专业文章。
- **现金流来源**：外贸订单佣金。

### 第二阶段：3-9个月——平台改造与引流（信任引擎）
- **目标**：mdlooker改造为PPE垂直合规工具包，获取首批50个注册用户、10个付费用户。
- **动作**：
  - 开发PPE合规自检工具（P0）、DoC生成器（P1）。
  - 搭建法规知识库（5-8个核心品类）。
  - 实施VIP免费赠送策略，替代冷开发信获客。
- **现金流来源**：外贸佣金为主，SaaS年费为辅。

### 第三阶段：9-18个月——服务闭环（价值放大器）
- **目标**：验证并跑通代办服务，完成首单“合规→生产”转化。
- **动作**：
  - 与欧代/美代/公告机构服务商建立合作关系。
  - 推出CE/UKCA/FDA代办服务包。
  - 平台上线AI问答助手、法规追踪功能。
- **现金流结构**：外贸佣金 + 合规代办费 + SaaS年费。

### 第四阶段：18个月以上——数字化壁垒（护城河深化）
- 开发客户专属文档协作平台（付费用户专享）。
- 利用累积数据优化AI问答与自检工具。
- 审慎拓展邻近品类（劳动防护用品）。

## 4. 核心竞争力构建
1. **能力三重奏**：英语专八(全球无障碍沟通+原文法规解读) + 计算机硕士(独立开发平台与AI工具) + 家族23年PPE供应链(深度行业认知)。
2. **供应链闭环**：从合规服务到生产订单的无缝衔接，客户粘性极高。
3. **数据飞轮**：平台累积的法规数据与用户行为数据，持续反哺产品优化。
4. **轻资产高毛利**：一人公司模式，先收款后服务，毛利率80-90%。

## 5. 风险控制与绝对避坑
| 绝对不做 | 原因 |
|---------|------|
| 开发公开SaaS与Euverify正面竞争 | 获客成本过高，资本驱动型对手已占先机 |
| 自己申请欧代/美代资质 | 需欧盟实体与法律责任，一人公司无法承担 |
| 初期拓展PPE以外品类 | 缺乏行业认知，稀释核心优势 |
| 追求“平台化”定位 | 与Alibaba等巨头竞争，无流量优势 |
| 融资/扩张团队 | 轻资产高毛利模式核心优势在于“轻” |
